## CoreAlign visual system

- Bright scientific workbench layout with a controlled reading width and clear section hierarchy
- Pure white light mode with navy, blue, cyan, yellow, coral, mint, and violet accents
- System font stack for reliable static deployment
- Lottie motion limited to one supportive rotate-then-crop cue and disabled with reduced motion preferences
- Accessible contrast, visible keyboard focus, semantic landmarks, and English copy
