# Tutorial version 1 archive

Do not follow this version for a new run. It was archived after a real user test loaded a `12 x 8`, `1.0 mm` config and then changed only the grid dimensions in the QuPath advanced dialog. The mixed values produced 54 present and 72 missing positions.

Use `tutorial/README.md` and the current release instead. The current workflow blocks duplicate configs, disables mixed-value advanced dialogs, shows a preflight summary, and enforces structural QC before approval or orientation.
