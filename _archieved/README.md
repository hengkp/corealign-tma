# Archived material

This folder contains files that are retained for provenance but must not be used for a production run.

- `legacy-multi-file-workflow` contains the old numbered Groovy workflow. The only supported runner is `workflow/CoreAlign.groovy`.
- `tutorial-v1` contains the first tutorial source. It did not make config path and preset verification explicit enough.
- `chatgpt-sites-scaffold` contains unused scaffolding from an earlier hosting approach. The live website uses Next.js and GitHub Pages.

The `_archieved` spelling is intentional so the repository matches the requested archive folder name.
